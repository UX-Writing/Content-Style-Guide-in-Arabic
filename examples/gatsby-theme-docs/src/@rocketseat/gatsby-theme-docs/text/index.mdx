# المقدمة

أهلًا بك في دليل أسلوب كتابة المحتوى (Content Style Guide) - الإصدارة الأولى، هذا الدليل (مفتوح المصدر) مساهمة لإثراء مجال كتابة تجربة المستخدم (UX Wriitng) باللغة العربية، يمكنك المساهمة معنا في إضافة أو تعديل وكذلك تحسين الدليل بإتباع التعليمات على Github.

## كيف تستخدم هذا الدليل؟

- للأهمية اقرأ المقال التالي حتى تكون ملمًًا بوظيفة الدليل [دليل أسلوب كتابة المحتوى وأهميته في كتابة تجربة المستخدم](https://uxwritingar.com/ux-writing/11992)
- يمكنك البدء في استخدام هذا الدليل بالاطلاع على الأقسام في القائمة الرئيسية وتطبيق ما يلزم على مشروعك.
- يتم تحديث الدليل باستمرار، استعرض سجّل التغييرات الأخيرة - [سجل التغييرات](https://github.com/UX-Writing/Content-Style-Guide-in-Arabic/releases/)
- إذا أعجبك الدليل شاركه مع زملائك في العمل وعلى حسابتك في شبكات التواصل الاجتماعي.
